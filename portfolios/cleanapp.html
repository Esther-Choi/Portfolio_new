클린앱(Clean App)은 설치된 악성 청소 앱들을 찾아 보여주고 쉽게 삭제 할 수 있도록 도와주는 청소 어플입니다. 나도 모르게 설치된 악성 앱을 확인 할 수 있으며 삭제 버튼 클릭 시 삭제 페이지로 이동되어 번거로움 없이 바로 삭제할 수 있습니다.

할머니의 핸드폰을 봐드리다가 나이 드신 분들은 이런 앱을 따로 찾고 확인하는데 어려움이 있어 그대로 불편함을 겪으시는걸 확인하고 이를 도와드리고자 개발한 어플입니다.

React Native를 사용하여 처음부터 끝까지 개발하였습니다.
